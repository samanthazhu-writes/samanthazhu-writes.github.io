# Samantha Zhu — Writer / Interactive Narrative Designer

This is the static personal homepage for Samantha Zhu.

## Purpose

The site introduces Samantha Zhu as a writer and interactive narrative designer, presents selected works, and provides a formal entry point to the dedicated Adèle Archive project portal.

## Files

- `index.html` — main homepage
- `css/style.css` — Morandi pink editorial portfolio styling
- `js/main.js` — mobile navigation behavior
- `.nojekyll` — keeps GitHub Pages static hosting simple

## Deployment

Upload all files in this folder to the root of the `samanthazhu-writes.github.io` repository.
